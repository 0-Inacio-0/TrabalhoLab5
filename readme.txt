O SGA não aceita arquivos com mais de 5mb então tive que retirar os executaveis. Por isso favor baixar do git https://github.com/0-Inacio-0/TrabalhoLab5.

Neste trabalho deverá ser desenvolvido um sistema para uma bolsa de valores qualquer, como a Bovespa,
utilizando o Nats(https://nats.io/) e Go(https://golang.org/).

Nas pastas Broker e Exchange estão os codigos.

Para executar este trabalho siga os seguinte passos:

    1-execute msgQueue.exe(servidor da fila de msg)
    2-execute exchange.exe(bolsa de valores)
    3-execute broker.exe(broker)

Pronto agora é só fazer as operações no broker.
